Calculator App


This project is a simple Calculator App built using HTML, CSS, and JavaScript. It provides an interactive interface for performing basic arithmetic operations like addition, subtraction, multiplication, and division.


Features

User-Friendly Interface: Responsive layout with clear buttons.

Core Arithmetic Functions: Supports addition, subtraction, multiplication, and division.


Additional Features:

AC: Clears the display.

+/-: Toggles the sign of the current input.

%: Calculates percentages.

Hover Effects: Buttons visually respond to user interaction.


File Structure

index.html: Defines the structure of the calculator.

style.css: Styles the layout, buttons, and effects.

app.js: Implements the calculator logic and button interactions.


How to Run

Clone or download the repository.

Open index.html in any web browser.

Use the buttons to perform calculations.

Notes

The app uses eval() to compute expressions, suitable for this context but not recommended for production.

Designed primarily for desktop browsers; minor adjustments may be needed for mobile.
